# 📊 Sales Data Insights Dashboard 🚀

## 🎯 Overview
This project analyzes a global sales dataset to uncover trends in revenue, customer behavior, and product performance. The dashboard provides actionable insights into sales channels and customer satisfaction levels.

## 🔑 Key Insights
* **Sales by Channel:** Comparing Online vs. Store performance (Online dominates with 93.33%).
* **Customer Segmentation:** Analysis of Loyal, Retail, and New customers.
* **Regional Performance:** Top cities by sales volume (Riyada, Block, etc.).
* **Demographics:** Sales distribution by Gender and Month trends.

## 🛠️ Tools Used
* **Power BI:** Advanced Visualization & DAX.
* **Power Query:** For data transformation.
